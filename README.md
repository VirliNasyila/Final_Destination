# College Database Project

This repository contains scripts and resources for a college-level database system project. The goal is to design and implement a relational database with robust data modeling and management features.

## Project Status

**Current Progress:**
- Data Models: ✔️ Completed
- DDL (Data Definition Language): ✔️ Completed

**Pending Work:**
- Functions
- Procedures
- Triggers
- Views
- DML (Data Manipulation Language)
- Additional documentation and testing

## Structure

- `models/` — Entity-relationship diagrams and schema definitions
- `ddl/` — SQL scripts for creating tables and constraints
- `dml/` — Scripts for inserting, updating, and deleting data (to be added)
- `procedures/` — Stored procedures (to be added)
- `functions/` — User-defined functions (to be added)
- `triggers/` — Database triggers (to be added)
- `views/` — SQL views (to be added)

## Getting Started

1. Clone this repository.
2. Review the data models and DDL scripts in their respective folders.
3. Pending features will be added as development continues.
